# Инструкция по развертыванию текстового редактора

## Что входит в проект

- Исходный код приложения (Vue 3 + TypeScript + Vite)
- Конфигурационные файлы проекта
- Документация

## Требования

- Node.js версии 18 или выше
- npm (устанавливается вместе с Node.js)

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

Эта команда установит все необходимые библиотеки для работы приложения.

### 2. Запуск в режиме разработки

```bash
npm run dev
```

После этого приложение будет доступно по адресу: http://localhost:5173

### 3. Сборка для продакшена

```bash
npm run build
```

Готовые файлы появятся в папке `dist/`. Эти файлы можно разместить на любом веб-сервере.

### 4. Предварительный просмотр продакшен-сборки

```bash
npm run preview
```

## Развертывание на сервере

После сборки (`npm run build`) содержимое папки `dist/` нужно:

1. Загрузить на ваш хостинг
2. Настроить веб-сервер (Nginx, Apache) на обслуживание статических файлов
3. Все маршруты должны перенаправляться на `index.html` для корректной работы SPA


## Основные функции редактора

- Многостраничное редактирование (формат A4)
- Форматирование текста
- Настройка шрифтов и цветов
- Выравнивание текста
- Списки
- Колонтитулы
- Настройка полей страницы
- Ориентация страницы (книжная/альбомная)
- Экспорт в PDF, DOCX, HTML, TXT
- Печать документов
- Отмена/повтор действий
