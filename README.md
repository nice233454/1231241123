# Текстовый редактор

Веб-приложение для создания и редактирования текстовых документов с поддержкой многостраничного формата.

## Возможности

- Многостраничное редактирование документов (формат A4)
- Форматирование текста (жирный, курсив, подчеркивание, зачеркивание)
- Настройка шрифта, размера и цвета текста
- Выравнивание текста (по левому краю, по центру, по правому краю, по ширине)
- Списки (маркированные и нумерованные)
- Верхние и нижние колонтитулы
- Настройка полей страницы (левое, правое, верхнее, нижнее)
- Ориентация страницы (книжная/альбомная)
- Экспорт в PDF, DOCX, HTML и TXT
- Печать документов
- Отмена и повтор действий
- Вставка текста из буфера обмена
- Автоматическое создание новых страниц при переполнении

## Технологии

- Vue 3
- TypeScript
- Vite

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект в режиме разработки:
```bash
npm run dev
```

3. Сборка для продакшена:
```bash
npm run build
```

4. Предварительный просмотр сборки:
```bash
npm run preview
```
